Projects

I am involved in a few programming projects, part or not of my PhD program. All of these projects are developed with the git versioning system and are free softwares under GPL licence.
  • CfgParser
    A C++ library helper tool for ini format data file parsing. It is a OO object code compatible with the python ConfigParser.
    The full code can be found and downloaded here.

  • Arbor
    A C++ implementation of a Particle Flow Algorithm (PFA) in the PandoraSDK framework for the next generation of e+e- linear collider generation. Based on the idea that electromagnetic and hadronic showers follow a tree topology, the Arbor algorithm aims to connect objects (calo hits) in order to cluster them in a tree topology.
    Initial idea from H. Videau and M. Ruan.
    The full code can be found and downloaded here.
