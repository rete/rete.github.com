Projects

I am involved in a few programming projects, part or not of my PhD program. All of these projects are developed with the git versioning system and are free softwares under GPL licence.
  • CfgParser
    A C++ library helper tool for ini format data file parsing. It is a OO object code compatible with the python ConfigParser.
    The full code can be found and downloaded here.

  • Baboon
    A C++ library for new generation of CALICE calorimeters (high granularity). It provides analysis tools for data processing, event monitoring, detector interface and implementations, etc ... Most of the code of this package will be forwarded to the new project Calipso (see below) after a complete further reading.
    The whole code is available here.

  • Calipso
    A C++ library and tools designed for CALICE calorimeter detectors. This is developed with the CALICE collaboration . It has been started very recently, is under development and under frequent releases. It should provide :
    • An online-monitoring for test beam
    • An offline-monitoring for event reconstruction or a simple data analysis
    • A data base condition interface
    • An electronics plug in for condition data base interface and data analysis
    • A full simulation modules for all CALICE calorimeter prototypes
    • A set of utils for HEP computing
    All of these modules should usable and flexible enough for any CALICE calorimeter.
    All the source code is available in its github repository
