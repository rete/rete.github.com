DQM4HEP

Project repositories

DQM4HEP is a C++ framework for online data quality analysis in the High Energy Physics domain. It aims to decouple the way data comes from detectors, are analyzed by physicists, and are viewed on graphical user interfaces. These different parts are run in independent client or server processes linked via a networking API.

The software provides the following functionnalities :

  • Event distributed system using server/client paradigm
  • Set of interfaces for data quality analysis , adapted to DQM purpose
  • Monitor element (histos, graphs, etc ...) distributed system
  • Visualization interfaces (Qt Gui)
  • Large scale remote process management
  • Generic IO support for physics analysis
  • From single detector prototype design up to full size HEP experiment
  • ELog interface

The project has its own main page, is free and under the terms of the GPL licence.

ArborPFA

Project repositories

A C++ implementation of a Particle Flow Algorithm (PFA) in the PandoraSDK framework.

Based on the idea that electromagnetic and hadronic showers follow a tree topology in sampling calorimeters, the ArborPFA algorithm aims to connect energy deposits to cluster them in a tree topology. This reconstruction algorithm is generic enough to be applied in full size high granular detectors such HL-LHC, ILC, CLIC or CEPC detectors.

The project provides a set of packages for :

  • Algorithms implementation (ArborContent)
  • Marlin (ILC specific) processor to run the algorithms (MarlinArbor)
  • An all-in-one installation package (ArborPFA)

Initial idea from H. Videau and M. Ruan.

The full code is free and under the terms of the GPL licence.

xdrstream

Project repository

A C++ library to read/write raw data in xdr format with different device implementation (file, buffer, socket, user defined). The main feature of this library is the decoupling between which data to serialize, how to serialize data and where to write data. Serialized data are organized in so-called records in which user defined blocks of data are written.

The full code is free and under the terms of the GPL licence.

CfgParser

Project repository

A C++ library helper tool for ini format data file parsing compatible with the python ConfigParser. It is designed to be as explicit and short-syntaxed as its python implementation.

Example of configuration file : 

# myfile.cfg

# A comment !

# A start of section
[MySection]

# A simple int value
MyIntValue = 42

# list of values are semi-colum separated
MyListOfIntValues = 12:45:32:78:4589

[AnotherSection]

AStringValue = A short or long description of whatever you want ...

One can read it simply with : 

CfgParser parser;

// read a single file
parser.read("myfile.cfg"); 

// get value using section name and key name !
int value = parser.getValue< int >("MySection", "MyIntValue");

// get a list of values !
std::vector< int > intList = 
	parser.getValue< std::vector< int > >("MySection", "MyListOfIntValues");

The full code is free and under the terms of the GPL licence.
